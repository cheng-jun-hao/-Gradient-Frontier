# 梯度前沿 Gradient Frontier — 实现计划

## 项目概述

构建一个名为"梯度前沿 Gradient Frontier"的AI资讯静态网站，复刻小米MIMO官网极简AI高级设计风格，部署于GitHub Pages。

## 技术栈

- 原生 HTML + CSS + JavaScript (ES Modules)
- Vite 6.x 构建工具
- JSON 数据驱动内容
- 纯 CSS 自定义变量实现主题切换

## 实现步骤

### Step 1: 项目初始化
- 使用 Vite 初始化项目（vanilla 模板）
- 配置 `vite.config.js`（base 路径、构建输出）
- 安装依赖
- 创建项目目录结构

### Step 2: 数据层搭建
- 创建 `src/data/news.json`，包含分类定义 + 示例资讯数据（约10条）
- 定义完整数据结构（id/title/summary/content/tags/source/date/featured）

### Step 3: CSS 主题与基础样式
- `src/styles/theme.css` — 浅色/深色 CSS 变量定义
- `src/styles/main.css` — 全局重置、字体引入、基础排版
- `src/styles/animations.css` — 滚动动画 + 过渡效果定义

### Step 4: 导航栏组件
- `src/styles/nav.css` + `src/components/nav.js`
- 固定悬浮置顶，半透明毛玻璃背景
- 站点 Logo + 导航链接（首页/归档） + 主题切换按钮
- 移动端汉堡菜单折叠

### Step 5: Hero 区域（核心视觉）
- `src/styles/hero.css` + `src/components/hero.js`
- 双层错位文字矩阵水印背景（"GRADIENT FRONTIER" / "梯度前沿" 重复排列）
- 鼠标跟随圆形光影（radial-gradient 动态更新 CSS 变量）
- 视差偏移效果（两层文字不同速率移动）
- 快速移动拉伸效果（事件间隔推断速度 → scaleX）
- 悬停标题切换文字明暗

### Step 6: 资讯卡片组件
- `src/styles/cards.css` + `src/components/newsCard.js`
- 极细浅边框 + 磨砂玻璃质感
- 悬浮上浮动画（translateY + box-shadow）
- 滚动渐入加载（IntersectionObserver）
- 交错延迟瀑布流效果

### Step 7: 搜索与分类筛选
- `src/components/search.js` — 实时搜索（input + 防抖200ms）
- `src/components/filter.js` — 标签横向滚动筛选
- 搜索 + 分类组合过滤

### Step 8: 资讯详情模态框
- `src/components/modal.js`
- 点击卡片弹出详情（标题/来源/正文/标签/日期）
- 背景遮罩 + 关闭按钮 + ESC 键关闭

### Step 9: 归档页
- `archive.html` + `src/styles/archive.css` + `src/components/archive.js`
- 按年月分组展示所有历史资讯
- 时间线式排版

### Step 10: 工具函数与动画系统
- `src/utils/animations.js` — IntersectionObserver 封装
- `src/utils/helpers.js` — 日期格式化、防抖、排序

### Step 11: 主题切换管理
- `src/components/theme.js`
- localStorage 持久化
- 系统偏好检测 (prefers-color-scheme)
- 平滑过渡动画

### Step 12: 响应式适配
- 移动端：卡片单列、导航折叠、Hero 简化
- 平板端：卡片两列
- 触摸设备交互优化

### Step 13: 站点图标与收尾
- `public/favicon.svg` — 自定义 SVG 图标
- 最终测试与优化
- 配置 GitHub Pages 部署说明
