# 梯度前沿 Gradient Frontier — 产品需求文档 (PRD)

## 1. 产品概述

"梯度前沿 Gradient Frontier" 是一个专注于AI领域前沿科技资讯的静态网站，每日分享AI行业最新动态与深度解读。

- **目标**：打造一个视觉精致、阅读体验优秀的AI科技资讯站点
- **用户**：站长本人（单人维护），读者为AI/科技爱好者
- **部署**：GitHub Pages，纯静态，无后端
- **内容管理**：通过编辑 JSON 数据文件增删改资讯内容

## 2. 核心功能

### 2.1 功能模块

1. **首页**：Hero 视觉区 + 资讯卡片流 + 分类/标签筛选 + 搜索
2. **归档页**：按月份组织的历史资讯浏览
3. **全局功能**：深色/浅色主题切换、响应式布局、滚动动画

### 2.2 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 `/` | 导航栏 | 固定悬浮置顶，站点名称 + 导航链接 + 主题切换按钮 |
| 首页 `/` | Hero 区域 | 双层错位文字矩阵水印背景，鼠标跟随圆形光影视差交互，鼠标快速移动时轻微拉伸，悬停标题切换明暗 |
| 首页 `/` | 分类/标签栏 | 横向标签筛选，点击切换资讯分类（大语言模型/CV/机器人/AI应用/前沿论文/AI行业等） |
| 首页 `/` | 搜索栏 | 实时关键词搜索，匹配标题、摘要、标签 |
| 首页 `/` | 资讯卡片流 | 极细浅边框卡片，轻微悬浮上浮动画，磨砂玻璃质感，滚动渐入加载动画 |
| 归档页 `/archive` | 月份分组 | 按年月分组展示所有历史资讯 |
| 资讯详情模态 | 详情弹窗 | 点击卡片展开完整资讯详情（标题、来源、摘要全文、标签、日期） |

## 3. 核心流程

### 3.1 读者浏览流程

1. 访问首页 → 看到 Hero 视觉区 + 最新资讯卡片流
2. 可通过分类/标签筛选感兴趣的内容
3. 可通过搜索栏关键词检索
4. 点击资讯卡片查看详情
5. 点击归档链接查看历史资讯

### 3.2 站长内容维护流程

1. 在 `src/data/news.json` 中添加新的资讯条目
2. 填写 id、标题、摘要、标签、来源、日期等字段
3. 运行 `vite build` 构建
4. 推送至 GitHub 部署

## 4. 用户界面设计

### 4.1 设计风格 — 复刻小米 MIMO 极简AI高级风格

**色彩体系：**
- 浅色主题：纯白 `#FFFFFF` 背景 + 低饱和银灰 `#F5F5F7`/`#E8E8ED` 辅助色
- 深色主题：炭黑 `#0A0A0B` 底色 + 冰蓝弱渐变光影（`#1A2332` → `#0D1520`）
- 强调色：冰蓝 `#4A9EFF`（浅色）/ `#6BB5FF`（深色），克制使用
- 禁止高饱和霓虹色

**字体：**
- 纤细无衬线字体，推荐 "Inter"（英文）+ "Noto Sans SC"（中文）
- 标题：`font-weight: 300-400`，大字号
- 正文：`font-weight: 300-400`，舒适行高 1.7-1.8

**布局：**
- 大量留白，宽松栅格居中排版（`max-width: 1200px`）
- 无多余装饰元素，信息层级简洁清晰

**卡片设计：**
- 极细浅边框（`1px solid rgba(0,0,0,0.06)`）
- 轻微悬浮上浮动画（`translateY(-2px)` + 阴影过渡）
- 弱磨砂玻璃质感（`backdrop-filter: blur(10px)`）

**导航栏：**
- 固定悬浮置顶，半透明背景 + 毛玻璃
- 极简圆角按钮样式

**动画：**
- 滚动时元素平滑渐入加载（Intersection Observer + CSS transition）
- Hero 区域鼠标跟随光影视差交互
- 页面过渡流畅自然

### 4.2 Hero 区域详细设计

- **文字矩阵水印背景**：重复排列 "GRADIENT FRONTIER" / "梯度前沿" 文字，形成双层错位矩阵
- **鼠标跟随光效**：圆形径向渐变遮罩跟随鼠标移动，照亮附近文字
- **视差效果**：文字层随鼠标位移产生轻微视差偏移
- **快速移动拉伸**：鼠标快速移动时光影产生轻微拉伸变形
- **悬停切换**：鼠标悬停标题区域时切换文字明暗显示

### 4.3 响应式

- Desktop-first 设计
- 移动端自适应：卡片单列、导航折叠、Hero 简化
- 触摸设备优化交互
